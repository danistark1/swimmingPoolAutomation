### swimmingPoolAutomation RasPi


- Automatic filter control (fixed and dual-speed pumps, 1 or 2 daily cycles) based on temperature.
- Chlorine/Temp/PH Sensors
- Chlorine dispenser
- Ph+/- dispenser
- Email notifications
